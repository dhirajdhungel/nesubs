### **Responsive Behavior:**

```
┌─────────────┐
│  [HEADER]   │  ← Full width
│             │
│  [CONTENT]  │  ← Stacks vertically
│             │
│  [BOX]      │  ← Full width
│             │
│  [BUTTON]   │  ← Full width
│             │
│  [FOOTER]   │  ← Full width
└─────────────┘
```

### **Mobile Optimizations:**
- ✅ Font sizes remain readable (min 16px)
- ✅ Buttons are touch-friendly (min 44px height)
- ✅ Tables stack on small screens
- ✅ Padding adjusts for mobile
- ✅ Images scale proportionally
- ✅ No horizontal scroll

## 🎯 VISUAL HIERARCHY

### **Priority Levels:**

**Level 1 - Most Important:**
- OTP codes (48px)
- Product codes (32px)
- Order totals (20px)
- Payment amounts (48px)

**Level 2 - Important:**
- Headlines (28-32px)
- Section titles (18px)
- CTA buttons (16px bold)

**Level 3 - Supporting:**
- Body text (16px)
- Labels (14px)
- Helper text (14px)

**Level 4 - Metadata:**
- Small print (12px)
- Footer text (11px)
- Timestamps (14px)
